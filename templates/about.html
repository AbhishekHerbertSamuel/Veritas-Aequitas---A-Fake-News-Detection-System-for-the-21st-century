About the Model:

This page describes the methodology used in the fake news detection model.

Data Source:

We have utilized the ISOT Fake News Dataset, which includes a total of 44,940 entries. This dataset is split into two parts: 21,417 entries in the True.csv file representing true news, and 23,523 entries in the Fake.csv file representing fake news.

Scope of the Data:

The data primarily covers international politics from the period 2016-2017 with a specific focus on the United States. This timeframe and geographical focus provide a rich dataset for understanding the dynamics of misinformation.

Model Architecture:

Our model employs the Multinomial Naive Bayes algorithm combined with a TF-IDF Vectorizer to classify news articles as 'true' or 'fake.' This method is well-suited for handling and analyzing text data and worked with 94% accuracy.

Technology Stack:

The back-end of our system is powered by Flask, a Python web framework. The front-end is built using HTML, CSS (enhanced with Bootstrap), and JavaScript, providing a responsive and user-friendly interface.

Integration Method:

The trained model is integrated with the front-end through the use of Pickle, a Python module that serializes and deserializes a Python object structure. This allows for the model to be easily loaded and utilized in the Flask application.
